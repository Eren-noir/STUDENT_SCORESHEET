# Student Scoresheet Project

This project is a simple Java program that collects and processes the marks of 15 students.  
The system allows the user to enter school details and marks for six subjects, then it calculates the total marks, averages, and the final grade for each student.

---

## What the Program Does

1. Collects school information:
   - School name
   - Teacher’s name
   - Grade
   - Year of study

2. Collects the names and marks of 15 students for the following subjects:
   - English
   - Math
   - History
   - Geography
   - Science
   - Programming

3. For each student, the program calculates:
   - Total marks
   - Final grade (A, B, C, D, or F)

4. For the whole class, the program also displays:
   - Total marks for each subject
   - Average marks for each subject
   - How many students got each grade (grade distribution)
